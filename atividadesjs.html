O que é JavaScript?

O JavaScript foi criado para dar interatividade às páginas web. Ele permite a execução de scripts no navegador, mas também pode ser executado no servidor. O conceito de "sandbox" no navegador impede que o JavaScript acesse certos recursos do sistema.
