Diferença entre Internet e Web

A Internet é uma rede global de computadores interconectados, permitindo a comunicação e o compartilhamento de dados. Já a Web é um serviço da Internet, que consiste em um conjunto de páginas interligadas acessadas por meio de navegadores.

A História da Internet e da Web

Década de 1960: A Internet começou como ARPANET, um projeto do governo dos EUA para comunicação entre universidades e órgãos militares.

Década de 1990: A World Wide Web foi criada por Tim Berners-Lee, revolucionando a forma de acessar e compartilhar informações.

Principais Marcos

  • 1991: A Web se torna pública.
  • 1994: Surge o primeiro navegador gráfico.
  • 2000: Explosão do uso de blogs e mídias sociais.

Protocolo HTTP

O que é o Protocolo HTTP?

O HTTP (HyperText Transfer Protocol) é um protocolo de comunicação utilizado na Web. Ele define como as requisições e respostas entre cliente (navegador) e servidor devem ser realizadas.

Exemplo de requisição HTTP:
GET /index.html HTTP/1.1
Este comando solicita o arquivo "index.html" ao servidor.

Campos de um Request

  • Method: Tipo de requisição (GET, POST, etc.)
  • URL: O caminho do recurso solicitado.
  • Headers: Informações adicionais como tipo de conteúdo, cookies, etc.

Exemplo de Resposta

HTTP/1.1 200 OK
Content-Type: text/html
Content-Length: 1234
